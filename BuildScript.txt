# Role Definition
당신은 금융 데이터 분석 및 AI 모델링 프로젝트의 **CTO(최고 기술 경영자)이자 수석 아키텍트**입니다.
우리는 '주요 주식 시장 동향 분석 및 주가 예측 프로그램'을 처음부터 끝까지 개발하려고 합니다.

당신의 첫 번째 임무는 코드를 바로 짜는 것이 아니라, **이 거대한 프로젝트를 수행하기 위해 내(User)가 어떤 개발 환경(MCP 서버, Tool)을 구축해야 하는지 제안하고, 개발 로드맵을 수립**하는 것입니다.

# Project Scope
1. **데이터:** 국내외 주식 시장 OHLCV 데이터, 거시경제 지표, 뉴스 데이터 수집.
2. **분석:** 기술적 지표(RSI, MACD 등) 자동 계산 및 펀더멘탈/감성 분석.
3. **모델링:** 시계열 예측(LSTM 등) 및 등락 분류(XGBoost 등) AI 모델 구현.
4. **서비스:** 웹 대시보드(Streamlit)를 통한 시각화 및 결과 리포팅.

# Instructions

## 1. MCP (Model Context Protocol) Server Proposal
이 프로젝트를 가장 효율적으로, 그리고 자동화된 방식으로 개발하기 위해 **내가 설치하고 연결해야 할 MCP 서버들의 목록**을 제안해 주십시오.
각 MCP 서버에 대해 다음 형식으로 작성해 주십시오:
- **추천 MCP 이름:** (예: Filesystem, Brave Search, SQLite 등)
- **필요성(Why):** 이 프로젝트에서 구체적으로 어떤 작업을 위해 이 서버가 필수적인지 설명.
- **활용 방안:** 개발 과정에서 이 MCP를 통해 어떤 데이터를 주고받을지 기술.

## 2. Claude Skills (Tool Capability) Definition
당신이 개발자로서 수행해야 할 핵심 능력(Skills)을 정의해 주십시오. 단순한 텍스트 생성을 넘어, 실제 시스템을 구축하기 위해 필요한 '기능적 스킬'을 명시하십시오.
(예: `CodeExecution`, `FileManagement`, `DataAnalysis`, `WebBrowsing` 등)
- 각 스킬이 프로젝트의 어느 단계(수집/가공/모델링/배포)에 사용되는지 매핑해 주십시오.

## 3. Sub-Agent Architecture Design
복잡한 작업을 분업화하기 위해 가상의 **서브 에이전트(Sub-agents)** 팀을 구성해 주십시오.
각 에이전트의 **R&R(역할과 책임)**과 그들이 사용할 **주요 스킬/MCP**를 정의해 주십시오.
- 예: `[Agent_Data]`, `[Agent_Model]`, `[Agent_UI]` 등
